- Modern, responsive design
- Hospital services showcase
- Medical team profiles

- Neurology
- Cardiology
- Orthopedics
- 24/7 Emergency Care

2. Open `index.html` in your browser

This site is automatically deployed to GitHub Pages when changes are pushed to the main branch.
